# QR Code Generator

Desafio de desenvolver um gerador de QR Code com a opção de download e compartilhar o link utilizando uma biblioteca para gerar o QR code.

## Tecnologias 

Aqui estão as tecnologias que utilizei no projeto:

* HTML
* CSS
* JavaScript

## Serviços Utilizados

* Github
* Vercel
